# smart-expense-tracker
Smart Expense Tracker Dashboard - A dynamic expense tracking web application with real-time analytics, custom categories, dark mode, and CSV export using HTML, CSS, JavaScript, and Chart.js.


## 🚀 Features

- ✅ Add, Edit, and Delete expenses (CRUD operations)
- 📅 Date-based expense tracking
- 📊 Real-time analytics using Chart.js (Doughnut Chart)
- 🎨 Dynamic category color customization
- 🌙 Dark mode toggle
- 📆 Monthly expense summary
- 💾 LocalStorage data persistence
- 📤 Export expenses to CSV file
- 🏷 Custom category support

---

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Chart.js
- LocalStorage API
- CSV File Handling (Blob API)
